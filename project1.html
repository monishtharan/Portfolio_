// PROJECT OVERVIEW

This project focuses on predicting urban expansion patterns for metropolitan areas using advanced machine learning techniques. By analyzing historical satellite imagery and geographical data, we developed a model that can forecast urban growth up to the year 2035.

// KEY FEATURES

  • Processed 20+ years of Landsat 8 satellite imagery using Google Earth Engine
  • Implemented Random Forest classifier for land cover classification
  • Built Cellular Automata-Markov chain model for spatial prediction
  • Achieved 94.5% accuracy in predicting urban sprawl patterns
  • Created interactive visualization dashboard for city planners

// METHODOLOGY

The analysis involved multiple stages: data acquisition from GEE, preprocessing of spectral bands, feature extraction, model training using ensemble methods, and validation against ground truth data. The final model incorporates 15+ environmental and socio-economic factors.
