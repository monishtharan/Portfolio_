// PROJECT OVERVIEW

A real-time computer vision system for smart city surveillance that detects and tracks objects in video streams. The system can identify vehicles, pedestrians, and cyclists in real-time, providing valuable analytics for urban planning and traffic management.

// KEY FEATURES

  • Real-time object detection using YOLOv8 with 30+ FPS processing
  • Multi-object tracking across video frames
  • Vehicle classification (cars, trucks, motorcycles, bicycles)
  • Pedestrian detection and crowd density analysis
  • Heatmap generation for traffic pattern analysis
  • RESTful API for integration with existing systems

// METHODOLOGY

The system uses a custom-trained YOLOv8 model optimized for urban surveillance footage. Object tracking is achieved using Deep SORT algorithm. The backend is built with FastAPI for high-performance inference, with the model deployed on edge devices for low-latency processing.
